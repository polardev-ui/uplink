# Uplink

Uplink is a terminal-based public chat room with authentication, powered by Firebase. Install globally and use the `uplink` command from anywhere.

## Features
- Public chat room
- User registration and login (username, password)
- Real-time chat via Firebase

## Installation

1. Clone the repository or install directly from GitHub:

```
pip install git+https://github.com/polardev-ui/uplink.git
```

## Usage

```
uplink
```

Follow the prompts to register or log in, then chat in the public room.

## Requirements
- Python 3.8+
- See requirements.txt for dependencies
